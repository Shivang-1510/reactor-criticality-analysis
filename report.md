# Technical Report: Reactor Criticality Analysis

## Introduction
Criticality is a fundamental concept in nuclear reactor physics, determining whether a reactor can sustain a chain reaction.

## Mathematical Model
The neutron diffusion equation coupled with k-effective eigenvalue problem is solved numerically.

## Method
Finite difference method and power iteration were used to compute neutron flux and multiplication factor.

## Results
The model successfully predicts reactor state and flux distribution.

## Conclusion
This project demonstrates core principles of reactor physics and provides a foundation for advanced nuclear simulations.
